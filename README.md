## What is done here
 * Connect to MetaMask
 *  Create a Modal for the Token List
 *  Fetch and Display Token List from CoinGecko API
 *  Display Selected Token Image and Symbol in Swap Box
 *  Get Price
 *  Get Quote
 *  Set a Token Allowance
 *  Perform the Swap
